# 🌦️ Weather Data Analysis

## 📌 Objective
Analyze weather dataset to extract meaningful insights.

## 🔍 Tasks Performed
- Found unique wind speed values
- Counted clear weather occurrences
- Filtered records with wind speed = 4 km/h
- Identified null values
- Renamed columns
- Calculated mean visibility
- Calculated standard deviation of pressure

## 🛠️ Tools Used
- Python
- Pandas

## 📊 Key Insights
- Identified weather patterns
- Cleaned dataset for analysis
- Extracted statistical metrics
